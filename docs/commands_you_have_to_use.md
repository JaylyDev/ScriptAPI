# API Missing Features for Slash Commands
#### The following command features are not implemented in scripting API (as of 1.19.50).

- `/kick`
- `/gamemode`
- `/gamerule`
- `/weather`
- `/ability`
- `/setblock ... destroy`
- `/xp`
- `/tickingarea`
- `/replaceitem` partial
- `/clear` partial
- `@s[hasitem=]` partial
- `/function` partial
- `/loot` partial
- `/scoreboard` partial
- `/locate`
- `/damage` partial
